和雲端(AWS S3)同步

但是R 可能沒有好的客戶端套件

R 提供system來呼叫命令列程序

可傳遞命令列參數

並且接收程序的回傳值，或截取stdout

可用於檢查是否有錯

可利用AWS CLI自動將資料上傳S3

cmd <- sprintf("aws s3api put-object --bucket %s --key %s --body %s %s %s",
  bucket, key, path, md5base64, class)
retval <- system(cmd, return = TRUE)

回傳結果的JSON，也可以利用rjson來處理
